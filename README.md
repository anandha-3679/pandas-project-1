# pandas-project-1
Pandas Data Analysis – Dataset (520×6)

This project performs complete analysis using ONLY Pandas.

## Features
- Load dataset
- View structure
- Missing value analysis
- Summary statistics
- Unique counts
- Categorical & numerical separation
- Correlation (pandas only)
- Sorting and grouping examples

## How to Run
1. Place your dataset inside /data as dataset.csv
2. Run:
   python analysis.py
